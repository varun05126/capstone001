# Capstone001

Capstone001 is a full-stack web application developed as part of an academic capstone project. The project demonstrates practical implementation of backend development, frontend integration, and cloud deployment using modern web technologies.

**Live Application:** https://capstone001.onrender.com  
**GitHub Repository:** https://github.com/varun05126/capstone001

---

## Project Description

This application reflects real-world software development practices with a clean project structure, proper backend–frontend integration, and cloud deployment. It is suitable for academic evaluation and portfolio presentation.
